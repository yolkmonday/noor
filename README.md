# Noor

Aplikasi waktu solat untuk macOS. Tampil di menu bar dengan hitung mundur, notifikasi adzan, dan pelacakan solat.

![macOS](https://img.shields.io/badge/macOS-14%2B-blue) ![Apple Silicon](https://img.shields.io/badge/Apple%20Silicon-native-green)

## Fitur

- Waktu solat otomatis berdasarkan lokasi
- Hitung mundur di menu bar dengan berbagai format
- Notifikasi & suara adzan
- Pelacakan solat harian (Solatku)
- Kalender mingguan
- Pilihan 500+ kota di Indonesia

## Install

### Homebrew

```bash
brew tap yolkmonday/noor
brew install --cask noor
```

### Manual

Download DMG dari [Releases](https://github.com/yolkmonday/noor/releases), buka, lalu seret Noor ke folder Applications.

## Persyaratan

- macOS 14 (Sonoma) atau lebih baru
- Apple Silicon (M1/M2/M3/M4)
